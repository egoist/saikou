# saikou [![Build Status](https://travis-ci.org/egoist/saikou.svg?branch=master)](https://travis-ci.org/egoist/saikou)

> Random text meme from ACG world

## Install

```
$ npm install --save saikou
```

## Usage

```js
const saikou = require('saikou')

saikou()
//=> '画面太美我不敢看'

saikou()
//=> '我的麒麟臂又要发作了'
```
